# 🛰️ Astro Sentinel — Frontend

The React frontend for the **Astro Sentinel** space debris tracking and visualization system.

---

## Tech Stack

| Technology | Role |
|------------|------|
| React 19 + TypeScript | UI framework and type-safe components |
| Vite | Build tool and dev server |

---

## Project Structure

```
src/
├── assets/      # Static assets (images, SVGs)
├── App.css      # Styles for the root component
├── App.tsx      # Root component
├── index.css    # Global styles
└── main.tsx     # App entry point
```

---

## Getting Started

### Prerequisites
- Node.js v20+ and npm

### Install dependencies
```bash
npm install
```

### Run the dev server
```bash
npm run dev
```

App will be available at `http://localhost:5173`
